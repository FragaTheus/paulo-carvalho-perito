## [v2.0.0] — Junho 2026 ✅ Atual

### Lançamento da nova plataforma institucional

Reformulação completa do site com nova stack moderna, identidade visual renovada e integração de formulário de contato funcional.

#### O que foi entregue

- **Landing page** com cinco seções estruturadas:
  - **Hero** — apresentação principal com chamada para ação
  - **Sobre** — perfil profissional e diferenciais do perito
  - **Competências** — áreas de atuação e especialidades técnicas
  - **Prova Social** — casos atendidos e credenciais
  - **CTA / Contato** — formulário integrado + links diretos (WhatsApp e e-mail)
- **Formulário de contato** com envio real via **EmailJS** (sem backend)
- **Design responsivo** — mobile-first, adaptado a todos os dispositivos
- **Performance** — geração estática (SSG) via Next.js, build otimizado com Turbopack
- **Stack atualizada** — Next.js 16, React 19, Tailwind CSS v4, TypeScript
